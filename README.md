## Descrição

Este trabalho tem como objetivo analisar a complexidade de algoritmos
de busca e ordenação sobre vetores e listas encadeadas, implementados
na **linguagem C** seguindo os princípios de Tipos Abstratos de Dados (TAD).

## Métricas Avaliadas

- Tempo de execução
- Média dos tempos
- Desvio padrão
- Comparação de desempenho entre algoritmos
- Geração de tabelas e gráficos com resultados  

## Estruturas de Dados

- Vetor
- Lista Encadeada

## Algoritmos Implementados

**Busca:**
- Busca Sequencial
- Busca Binária

**Ordenação:**
- Bubble Sort
- Insertion Sort
- Selection Sort
- Quick Sort
- Merge Sort


## Medição de Tempo

A medição de tempo foi realizada com a função `clock()` da biblioteca
`<time.h>`, registrando o tempo de CPU para cada execução dos algoritmos.  

## Resultados

Os tempos de execução, médias e desvios padrão estão
documentados no PDF anexado.
